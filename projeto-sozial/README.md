# Projeto Social (Angular) — até a Aula 07

Este projeto contém:
- Componentes: Header, Footer, CampaignCard, CampaignFilter
- Páginas: Home, Sobre, Campanhas, Campanha Detalhe
- Rotas: /, /campanhas, /sobre, /campanhas/:id, wildcard
- Service: CampanhasService (mock) com getAll/getById/delete
- Diretivas: *ngIf com else, *ngFor com trackBy, ngClass
- Pipe custom: resumo

## Como rodar
1) Instale Node.js LTS
2) Instale Angular CLI (se necessário):
```bash
npm i -g @angular/cli
```
3) Instale dependências:
```bash
npm install
```
4) Rode:
```bash
npm start
```
Abra: http://localhost:4200
